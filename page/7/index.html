终止容器

docker container stop用来终止一个正在运行的容器，如果容器指定的app终止，那么容器也会立刻终止。
可以使用 docker container ls -a查看到终止的容器。
docker container start：将终止的容器重新运行。
docker container restart：将运行的容器终止，然后再重启。
